Your sketch:

Instructions

Note: This exercise assumes you remember how to create and use input controls, which was covered in PDM 1, Week 7 lecture 1.

Your task is to create a (very) simplified account creation form. The form should allow a user to type in their chosen username. If the username is taken, it should display an appropriate message, e.g. "Username is taken! Please try again." If the username is available, show a success message.

Here are the steps:

  1. In sketch.js, create a canvas at a size of your choosing.
  2. Create a global Set() to store usernames as they are added.
  3. Add a text input and a button to the canvas. Don't worry too much about making them look niceâ€”the goal is to practice using a Set.
  4. When the user clicks the button, get the username from the text input and:
    • ...if the text is not empty and the username is not already in the set, add it to the set and show a success message of your choosing.
    • ...if the text is not empty but the username is taken, show a message informing the user that the username is taken.
    • ...if the text is empty, show a message saying that the username can't be empty.

You can choose to print messages to the console instead of showing them on the canvas.

There are no checks for this exercise. Consider writing your own tests for step 4.

Expected output:

You will need to write a function to serve as an event handler for the button. The function should be called when the button is clicked. Used the mouseClicked() method to connect the button to the event handler function.

Use the value() method to get the text in the input.

The MDN Set documentation will be helpful. You will need to use two set methods: one to check if the username is already in the set and another to add the username.
