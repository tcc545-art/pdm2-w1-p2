2 Star - Exercise 1

Your sketch:

Instructions

sketch.js contains code to create a basic chart. You will add functionality that will allow the user to sort the data points by different properties.

Start by taking a look at the code already in sketch.js. The key things to look at are the Data class, which represents a data point (one column) for the chart, and draw(), which draws the Data objects as they appear in the columns array. The columns array is populated with random Data objects in the randomShapes() function.

Your task is to write custom sorting functions to enable the interactions described in the list below. See the hint for more guidance on sorting.

  • When the user presses "w", sort the columns array by the Data objects' width property.
  • When the user presses "h", sort the columns array by the Data objects' height property.
  • When the user presses "c", sort the columns array by the Data objects' colour property. You can decide what it means to sort by colour. In the video below, colour values were sorted by the red channel, then the green channel, then the blue channel. See the hint for a reminder of how to access colour channel values separately.
  • Optional extension: allow the user to toggle between sorting in ascending order and sorting in descending order.

Expected output:

For guidance on sorting an array, see the MDN sort() method. In particular, look at the section on defining a custom sort function, called compareFn in the MDN reference. To use a custom sort function, call it as follows:

arr.sort(compareFn);

To sort by colour, you will not be able to compare colours directly using comparison operators because p5.js color is an object. Instead, you should compare the red, green, and blue channels of each colour using the p5.js red(), green(), and blue() functions. For example: 

if (red(a.colour) < red(b.colour)) {
    return -1;
}
